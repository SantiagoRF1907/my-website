My Projects

This is one of my first projects using HTML, CSS and JS. It is a simple web page that creates a pattern with squares.

The app asks the user for an amount of shapes (squares) from 1 to 50, a color, an angle direction and an angle increment. The amount of shapes will determine how many squares will be displayed in the canvas area, the color input, the color of such squares and the angle increment will define the square's angle, adding the selected angle increment to each square in the pattern.
